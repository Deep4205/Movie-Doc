About Movie Doc

Movie Doc is a free movie documentation website. It can be used to review ratings, and movie related all information.

This website uses OMDb API as base ingredient. 

Movie Doc is developed using : Express.js, Node.js,  EJS templating, Bootstrap, HTML, CSS and JavaScript
